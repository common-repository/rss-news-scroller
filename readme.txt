RSS News Scroller is a simple WordPress plugin to create a list of news with images from RSS feed and display it in marquee

== Description ==

RSS News Scroller simple and lightweight news scroller which you can use by just activating it. Classic jQuery 3D scrolling effect attract user to read news from Feed.

= Features =
* News Scroller with RSS
* Cube Effect
* Scrolling Effects settings
* Widget

= Comming Soon =
* Multiple RSS Feeds
* Tell me your ideas

If you have suggestions for a new add-on, feel free to email me at admin@vhghorecha.in.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the RSS News Scroller menu that appears in your admin menu
